ІДЕЯ: код лежить у твоєму безкоштовному репозиторії на GitHub. Друг ставить
програму ОДИН раз. Далі щоразу при запуску вона сама підтягує з GitHub твої
свіжі зміни — пересилати нічого не треба.

(Назви .bat-файлів англійською навмисно — Windows-консоль псує кирилицю
всередині .bat, тому текст у них лишили латиницею. Це нормально.)

────────────────────────────────────────────────────────────────────
ЯК НАЛАШТУВАТИ (робиться один раз)
────────────────────────────────────────────────────────────────────
1) Подвійний клік:  «Step 1 - GitHub login.bat»
   • відкриється браузер, увійди в GitHub (або зареєструйся — безкоштовно);
   • встав код, який покаже вікно, і натисни Authorize.

2) Подвійний клік:  «Step 2 - Create repo.bat»
   • сам створить репозиторій, заллє туди файли, пропише адресу
     і збере архів  discord-autocompress-setup.zip

Усе. Більше нічого вручну робити не треба.

────────────────────────────────────────────────────────────────────
ЩО НАДІСЛАТИ ДРУГУ
────────────────────────────────────────────────────────────────────
Один файл:  discord-autocompress-setup.zip  (з'явиться в цій теці).
Друг:
   • розпакує його в будь-яку теку;
   • подвійний клік «Install.bat».
Далі інсталятор сам поставить Python, ffmpeg, завантажить програму з твого
GitHub, увімкне автозапуск і запустить. Готово.
(Якщо в друга не було Python — інсталятор його поставить і попросить
запустити файл ще раз. Просто клікнути вдруге.)

────────────────────────────────────────────────────────────────────
ЯК ВИПУСКАТИ ОНОВЛЕННЯ (коли щось зміниш у програмі)
────────────────────────────────────────────────────────────────────
Подвійний клік:  «Publish update.bat»
У друга оновиться само при наступному запуску (або він тисне «Update now.bat»).

Якщо ДОДАВ НОВИЙ файл програми — впиши його назву в manifest.json,
тоді «Publish update.bat».

────────────────────────────────────────────────────────────────────
БЕЗПЕКА
────────────────────────────────────────────────────────────────────
• У ТЕБЕ авто-оновлення НЕ вмикається — твої файли з GitHub не перезапишуться.
  Дозвіл дає файл-маркер «.autoupdate», який ставить лише інсталятор у друга.
• Репозиторій Public видно всім — не клади туди приватне (паролі, вебхуки).
